## Cellular Router

### GL-XE3000

![gl-xe3000 led](https://static.gl-inet.com/docs/router/en/4/user_guide/gl-xe3000/hardware_info/xe3000led.jpg){class="glboxshadow gl-70-desktop"}

* Power LED
	* Green: Power on
	* Off: Power off

* Battery LED (During charging)
	* Flashing Red: battery level 0%-20%
	* Flashing Violet: battery level 21%-50%
	* Flashing Green: battery level 51%-99%
	* Solid Green: battery level 100%

* Battery LED (Not charging)

	When not charging, the Battery LED lights up for 5 seconds after the power button is pressed.

	* Red: battery level 0%-20%
	* Violet: battery level 21%-50%
	* Green: battery level 51%-100%
	
* Internet LED
	* On: Internet is connected
	* Off: Internet is disconnected

* 2.4G Wi-Fi LED
	* On: 2.4G WiFi is readied
	* Flashing: 2.4G WiFi is communicating

* 5G Wi-Fi LED
	* On: 5G WiFi is readied
	* Flashing: 5G WiFi is communicating

* Cellular LED
	* Mobile Tower is conntected

	* Solid 4/4: battery level 76%-100%

* Power LED
	* Green: Power on

* WAN LED
	* On: WAN port is connected
	* Flashing: WAN port is communicating
	* Off: WAN port is disconnected

* LAN LED
	* On: LAN port is connected

	* Off: LAN port is disconnected

* Wi-Fi LED
	* On: 2.4G WiFi is readied

## Security Gateway & Mini Routers

### GL-MT2500/GL-MT2500A

![gl-mt2500 interface](https://static.gl-inet.com/docs/router/en/4/user_guide/gl-mt2500/hardware_info/mt2500_interface.jpg){class="glboxshadow"}

* System
	* Blue: System booting
	* Flashing blue: Searching network
	* White: Working normally
* VPN
	* Off: VPN client is not connected
	* Flashing: VPN client is connecting
	* Solid: VPN client is connected
	
	VPN server status not displayed on LED.
